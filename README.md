## Usage



## 1. use docker compose

```
git clone https://github.com/xx025/SnapSort.git
cd SnapSort
docker compose up -d
```



## 2. use source code
```
git clone https://github.com/xx025/SnapSort.git
cd SnapSort
uv sync
cd frontend
yarn install
yarn build
cd ..
uv run app.py
```
